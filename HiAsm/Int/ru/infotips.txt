Для разрыва связи между точками достаточно щелкнуть правой кнопкой мыши на одной из двух соединенных точек
Для прокрутки области построения схемы по горизонтале достаточно вращать колесико мышки, удерживая клавишу Ctrl
Для прокрутки всей области построения схемы нужно использовать среднюю кнопку мыши
Для показа дополнительных точек в компонентах нужно поставить галочку напротив соответствующей точки в панеле Свойства на вкладке Точки
Для быстрого снятия скриншота схемы нужно использовать сочетания клавиш Ctrl+Q
Для входа в составной элемент контейнер достаточно два раза щелкнуть по нему левой кнопкой мыши
Для быстрого получения справочной информации по компоненту достаточно выделить его и нажать сочетание клавиш Ctrl+F1
Для закрытия текущего проекта или вкладки достаточно щелкнуть по ней средней кнопкой мыши, либо выбрать пункт контекстного меню Закрыть, либо нажать крестик справа на панели вкладок
